Built with the Polymarket CLOB API

Every number on this page is live. Here's exactly what's being called — no black boxes, no wrappers. Tap any endpoint to read the full docs.

GET
gamma-api.polymarket.com/markets
Fetches active markets with prices, volume, categories, and token IDs
→ docs.polymarket.com/api-reference/events/list-events
GET
clob.polymarket.com/book?token_id=<id>
Fetches real-time order book depth (bids + asks) for any outcome token
→ docs.polymarket.com/api-reference/market-data/get-order-book
GET
clob.polymarket.com/midpoints?token_id=<id>
Retrieves midpoint price (avg of best bid + ask) for displayed probability
→ docs.polymarket.com/api-reference/data/get-midpoint-price
